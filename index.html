SimpleSearch GitHub Project Web Page

Container home for all of the associated reports and metric data for the SimpleSearch project on GitHub.
  • Valgrind: Specifically the memchecker tool, use to check for double frees, use after free, etc.
  • cppcheck: Static analysis checker for C/C++ syntax.
  • Doxygen: Source code documentation generated from code comments.
  • sloccount: Source Lines of Code count
  • gcov/lcov: Code Testing Coverage Report
