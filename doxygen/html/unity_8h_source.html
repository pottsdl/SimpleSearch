00018 // Integers
00019 //     - Unity assumes 32 bit integers by default
00020 //     - If your compiler treats ints of a different size, define UNITY_INT_WIDTH
00021 
00022 // Floats
00023 //     - define UNITY_EXCLUDE_FLOAT to disallow floating point comparisons
00024 //     - define UNITY_FLOAT_PRECISION to specify the precision to use when doing TEST_ASSERT_EQUAL_FLOAT
00025 //     - define UNITY_FLOAT_TYPE to specify doubles instead of single precision floats
00026 //     - define UNITY_FLOAT_VERBOSE to print floating point values in errors (uses sprintf)
00027 //     - define UNITY_INCLUDE_DOUBLE to allow double floating point comparisons
00028 //     - define UNITY_EXCLUDE_DOUBLE to disallow double floating point comparisons (default)
00029 //     - define UNITY_DOUBLE_PRECISION to specify the precision to use when doing TEST_ASSERT_EQUAL_DOUBLE
00030 //     - define UNITY_DOUBLE_TYPE to specify something other than double 
00031 //     - define UNITY_DOUBLE_VERBOSE to print floating point values in errors (uses sprintf)
00032 
00033 // Output
00034 //     - by default, Unity prints to standard out with putchar.  define UNITY_OUTPUT_CHAR(a) with a different function if desired
00035 
00036 // Optimization
00037 //     - by default, line numbers are stored in unsigned shorts.  Define UNITY_LINE_TYPE with a different type if your files are huge
00038 //     - by default, test and failure counters are unsigned shorts.  Define UNITY_COUNTER_TYPE with a different type if you want to save space or have more than 65535 Tests.
