Precondition:

Global Data:
  • g_debug_output - Debug output flag
Description:
Description of what the function does from the caller's point of view.
Algorithm:
Using the Work_Queue, waits to be signaled that there are file paths which are read to be searched for words. Once a path appears, thread pulls it off of the queue and starts reading and processing it. If the path is instead the "EXIT" command, then thread gracefullys exits to be harvested by thread_join.
